## Задание 3 — Круг (площадь равна половине площади экрана)

**Математическое обоснование:**

Площадь экрана:
```text
Sэкр = width * height
```

Площадь круга:
```text
Sкр = Sэкр / 2
```

Радиус круга:
```text
r = sqrt(Sкр / π)
```

В коде радиус вычисляется по формуле выше, после чего круг рисуется по центру экрана методом `drawCircle`.

## Задание 4 — Автомобиль

Автомобиль отрисован набором графических примитивов `Canvas`:

| Элемент | Метод |
|---------|-------|
| Корпус | `drawRect` |
| Колёса | `drawCircle` |
| Кабина | `drawArc` |
| Стойки/перегородка | `drawLine` |
| Ручка | `drawArc` |

Все координаты считаются относительно центра экрана, поэтому фигура остается по центру при разных размерах дисплея.

## Ответы на контрольные вопросы

**1. Что такое виджет в Android?**  
Виджет — это элемент интерфейса пользователя, наследник класса `View`, который отображает данные и/или обрабатывает действия пользователя (`TextView`, `Button`, `EditText` и т.д.).

**2. Что такое графические примитивы?**  
Графические примитивы — базовые элементы рисования (`линия`, `прямоугольник`, `круг`, `дуга`, `Path`), из которых строится изображение на `Canvas` с помощью `Paint`.

**3. Что такое переопределение метода?**  
Переопределение (`override`) — это реализация метода родительского класса в дочернем классе с собственной логикой. В работе переопределен метод `onDraw(canvas)` класса `View`.

**4. Что такое наследование?**  
Наследование — механизм ООП, при котором дочерний класс получает свойства и методы родительского класса. В работе `DrawView` наследуется от `View`.
